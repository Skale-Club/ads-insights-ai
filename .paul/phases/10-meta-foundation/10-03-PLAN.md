<objective>
## Goal
Add the platform selector to the dashboard, create the company registration UI, and create the Meta Ads connection wizard in Settings. After this phase, users can register their company and connect their Meta Ads account — and the TopBar shows a Google | Meta toggle.

## Purpose
This is the visible entry point for all Meta functionality. Without it, users have no way to connect Meta or switch to Meta dashboards.

## Output
- Platform selector in TopBar (Google Ads / Meta Ads)
- DashboardContext extended with `platform`, `metaAccounts`, `selectedMetaAccount`
- CompanySection in Settings
- MetaAdsSection in Settings (OAuth connect button + account selector)
</objective>

<acceptance_criteria>

## AC-1: Platform selector visible in TopBar
```gherkin
Given a logged-in user on any dashboard page
When they look at the TopBar
Then they see a Google Ads | Meta Ads segmented control or tabs
And clicking Meta Ads switches platform to 'meta' in DashboardContext
And clicking Google Ads switches back to 'google'
And the platform persists in localStorage as 'adsinsight:platform'
```

## AC-2: DashboardContext extended
```gherkin
Given the DashboardContext
When platform is 'meta'
Then selectedMetaAccount is available (null if no Meta account connected)
And metaAccounts lists all connected Meta ad accounts
And platform-switching calls setSelectedMetaAccount if first time
```

## AC-3: Company registration form
```gherkin
Given Settings → Company Profile section
When user fills name (required) + optional CNPJ, website, vertical
And clicks Save
Then it upserts into companies table via Supabase client
And shows success toast
```

## AC-4: Meta Ads connection wizard
```gherkin
Given Settings → Meta Ads section (visible after company is saved)
When user clicks "Connect Meta Ads"
Then they are redirected to Meta OAuth dialog URL:
  https://www.facebook.com/v20.0/dialog/oauth
    ?client_id=VITE_META_APP_ID
    &redirect_uri=VITE_META_REDIRECT_URI
    &scope=ads_read,ads_management,business_management
    &state={supabase_user_id}
    &response_type=code
And on return from OAuth (settings?meta_connected=true), account list loads
And user can select which ad account to use as default
And connected status shows: user name + account count + "Disconnect" button
```

## AC-5: Meta account selector in TopBar (when platform=meta)
```gherkin
Given platform === 'meta' and metaAccounts.length > 0
When the account dropdown renders in TopBar
Then it shows Meta ad accounts (not Google accounts)
And selecting one updates selectedMetaAccount in DashboardContext
```
