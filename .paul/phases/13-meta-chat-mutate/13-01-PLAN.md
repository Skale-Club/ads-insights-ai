  <name>Task 2: Implement meta-mutate edge function</name>
  <files>supabase/functions/meta-mutate/index.ts</files>
  <action>
    Mirror google-ads-mutate structure. Zod schema:
    ```ts
    z.object({
      accessToken: z.string().min(1),
      action: z.enum(["pauseCampaign", "enableCampaign", "updateDailyBudget", "updateLifetimeBudget"]),
      campaignId: z.string().optional(),
      adSetId: z.string().optional(),
      amountCents: z.number().positive().optional(), // Meta uses cents (currency * 100)
    })
    ```

    Meta API base: `https://graph.facebook.com/v20.0/`
    - pauseCampaign: `POST /{campaignId}?status=PAUSED&access_token={token}`
    - enableCampaign: `POST /{campaignId}?status=ACTIVE&access_token={token}`
    - updateDailyBudget: `POST /{adSetId}?daily_budget={amountCents}&access_token={token}`
    - updateLifetimeBudget: `POST /{campaignId}?lifetime_budget={amountCents}&access_token={token}`

    Return `{ success: true }` or `{ error: "...", code: N }` with Meta's HTTP status.
    Require campaignId for pause/enable/updateLifetimeBudget; adSetId for updateDailyBudget.
    Return 400 if required field missing.
  </action>
  <done>AC-2 satisfied</done>
</task>
