  <files>supabase/functions/meta-reports/index.ts</files>
  <action>
    Model after google-ads-reports/index.ts. Use `npm:zod@3.22.4`.

    **Request schema (Zod):**
    ```ts
    z.object({
      accessToken: z.string().min(1),
      accountId: z.string().regex(/^act_\d+$/, "accountId must be in format act_XXXXXXXXX"),
      reportType: z.enum(["overview", "campaigns", "adsets", "ads", "insights_by_placement", "daily_performance"]),
      startDate: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
      endDate: z.string().regex(/^\d{4}-\d{2}-\d{2}$/),
    })
    ```

    **Base URL:** `https://graph.facebook.com/v20.0/`
    **Date param format:** `time_range={"since":"YYYY-MM-DD","until":"YYYY-MM-DD"}`

    **Marketing API fields per report type:**

    `overview`:
    - `GET /${accountId}/insights?fields=spend,reach,impressions,clicks,ctr,cpc,cpm,actions,action_values,cost_per_action_type&time_range=...&level=account`
    - `roas` = action_values[action_type=offsite_conversion.fb_pixel_purchase].value / spend (or 0 if missing)

    `campaigns`:
    - `GET /${accountId}/campaigns?fields=id,name,status,objective,daily_budget,lifetime_budget,insights.fields(spend,impressions,clicks,ctr,actions,action_values,cost_per_action_type){time_range,...}&limit=200`

    `adsets`:
    - `GET /${accountId}/adsets?fields=id,name,campaign_id,status,targeting,daily_budget,start_time,end_time,insights.fields(spend,impressions,ctr,actions,action_values){time_range,...}&limit=200`
    - `targetingSummary` = stringify key targeting: age_min-age_max, top 2 geos, top 2 interests

    `ads`:
    - `GET /${accountId}/ads?fields=id,name,adset_id,status,creative{title,body,image_url},insights.fields(spend,impressions,ctr,cpc,actions,action_values){time_range,...}&limit=200`

    `insights_by_placement`:
    - `GET /${accountId}/insights?fields=impressions,clicks,spend,ctr,cpc&breakdowns=publisher_platform,platform_position&time_range=...&level=ad`
    - Group by publisher_platform+platform_position → placement label

    `daily_performance`:
    - `GET /${accountId}/insights?fields=spend,impressions,clicks,actions&time_range=...&time_increment=1&level=account`

    **Token refresh logic** (before making any API call):
    - Call Supabase (service role key from env) to check meta_connections.expires_at for the user
    - If expires_at < now + 7 days: call `GET /oauth/access_token?grant_type=fb_exchange_token&...`
    - Update meta_connections with new token

    Note: Token refresh requires knowing the user. Pass `userId: z.string().optional()` in schema (needed for refresh only). If not passed, skip refresh.

    Return `{ data: [...] }` on success.
